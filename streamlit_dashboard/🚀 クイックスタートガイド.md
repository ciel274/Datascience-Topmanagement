# 🚀 クイックスタートガイド

SPI対策ダッシュボードをすぐに使い始めるための手順です。

## 📦 ステップ1: ファイルの準備

ダウンロードしたファイルを以下のように配置してください：

```
streamlit_dashboard/
├── dashboard.py          ← メインのアプリケーション
├── requirements.txt      ← 必要なパッケージリスト
├── run.sh               ← macOS/Linux用実行スクリプト
├── run.bat              ← Windows用実行スクリプト
├── sample_log.csv       ← サンプルデータ
├── README.md            ← 機能説明書
├── SETUP_GUIDE.md       ← 詳細セットアップガイド
└── QUICK_START.md       ← このファイル
```

## 💻 ステップ2: ダッシュボードを起動

### **Windows ユーザー**

1. ダウンロードしたフォルダを開く
2. `run.bat` をダブルクリック
3. ブラウザが自動的に開き、ダッシュボードが表示されます

### **macOS/Linux ユーザー**

1. ターミナルを開く
2. ダウンロードしたフォルダに移動：
   ```bash
   cd /path/to/streamlit_dashboard
   ```
3. 実行スクリプトを実行可能にする：
   ```bash
   chmod +x run.sh
   ```
4. スクリプトを実行：
   ```bash
   ./run.sh
   ```
5. ブラウザが自動的に開き、ダッシュボードが表示されます

### **VSCode での実行**

1. VSCode でプロジェクトフォルダを開く
2. ターミナルを開く（`Ctrl+`` または `Cmd+``）
3. 以下を実行：
   ```bash
   # macOS/Linux
   source venv/bin/activate
   streamlit run dashboard.py
   
   # Windows (PowerShell)
   .\venv\Scripts\Activate.ps1
   streamlit run dashboard.py
   
   # Windows (Command Prompt)
   venv\Scripts\activate.bat
   streamlit run dashboard.py
   ```

## 🎯 ステップ3: ダッシュボードの設定

ダッシュボードが起動したら、サイドバーで以下を設定します：

### **1️⃣ 企業・目標設定**

サイドバーの「🏢 企業・目標設定」を展開して入力：

| 項目 | 例 | 説明 |
|------|-----|------|
| **志望企業名** | 〇〇商社 | 対策対象の企業 |
| **目標正答率** | 80% | SPI試験での目標 |
| **時間設定** | 標準 | 解答時間の基準 |

### **2️⃣ 学習データを入力**

サイドバーの「✏️ 学習データ入力」を展開：

1. **日付** - 問題を解いた日を選択
2. **科目** - 「非言語」または「言語」を選択
3. **ジャンル** - 科目に応じたジャンルが表示されます
4. **単元** - ジャンルに応じた単元が表示されます
5. **解答時間** - 実際にかかった時間（秒）
6. **正誤** - 「〇」（正解）または「✕」（不正解）
7. **ミスの原因** - 不正解の場合、原因を選択
8. **学習時間** - その問題に費やした学習時間（分）
9. **「➕ データを追加」ボタン** - クリックしてデータを保存

**例：**
- 日付：2024-11-18
- 科目：非言語
- ジャンル：推論
- 単元：集合の推論 (ベン図)
- 解答時間：110秒
- 正誤：〇
- ミスの原因：-
- 学習時間：15分

## 📊 ステップ4: ダッシュボードを確認

データを追加すると、メインエリアに以下が自動的に表示されます：

### **アクションカード**
- 次週の重点単元を表示
- 改善すべき分野を提示

### **主要指標（KPIカード）**
- **現在の正答率** - 期間平均の正答率
- **目標との差** - 目標に対する進捗
- **時間超過率** - 目標時間を超過した割合
- **総演習数** - 累計問題数

### **分析グラフ**
- **日別正答率トレンド** - 正答率の推移
- **達成度ゲージ** - 目標に対する進捗
- **優先単元 Top 5** - 改善が必要な単元
- **誤答原因分析** - ミスの原因分析

### **科目別達成状況**
- 各科目の正答率とプログレスバー

## 📁 ステップ5: ファイル管理

### **過去のデータをアップロード**

サイドバーの「📁 ファイル管理」を展開：

1. **学習ログCSV** - 過去の学習データをアップロード
2. **問題マスタCSV** - 独自の問題セットをアップロード

**CSVファイルの形式：**

学習ログ：
```csv
日付,問題ID,正誤,解答時間(秒),ミスの原因,学習投入時間(分)
2024-11-01,N-A01,〇,110,-,15
2024-11-01,N-A02,✕,120,時間不足,20
```

### **データをダウンロード**

「📋 入力済みデータ一覧」セクションで：

1. 入力したデータが表示されます
2. 「📥 CSVをダウンロード」ボタンをクリック
3. CSVファイルがダウンロードされます

## 🔍 ステップ6: 分析期間の変更

サイドバーの「🔍 分析期間」で：

1. **開始日** - 分析を開始する日を選択
2. **終了日** - 分析を終了する日を選択

ダッシュボードが自動的に更新され、選択期間のデータのみが表示されます。

## 💡 使用例

### **例1: 新規ユーザー**

1. `run.bat` または `./run.sh` を実行
2. 「志望企業名」に「〇〇商社」と入力
3. 「目標正答率」を80%に設定
4. 「✏️ 学習データ入力」で問題を追加
5. ダッシュボードで結果を確認

### **例2: 過去データがある場合**

1. `run.bat` または `./run.sh` を実行
2. サイドバーの「📁 ファイル管理」を展開
3. 「学習ログCSV」に過去のCSVをアップロード
4. ダッシュボードが自動的に更新
5. 「🔍 分析期間」で期間を調整して分析

### **例3: サンプルデータで試す**

1. `run.bat` または `./run.sh` を実行
2. サイドバーの「📁 ファイル管理」を展開
3. 「学習ログCSV」に `sample_log.csv` をアップロード
4. ダッシュボードが自動的に更新
5. グラフやKPIが表示されます

## ⚙️ よくある質問

### **Q1: ブラウザが自動的に開かない場合**

A: ターミナルに表示されたURLをブラウザにコピー&ペーストしてください。
例：`http://localhost:8501`

### **Q2: 「streamlit: command not found」エラーが出た場合**

A: 仮想環境が有効になっていません。以下を実行してください：
```bash
# macOS/Linux
source venv/bin/activate

# Windows (PowerShell)
.\venv\Scripts\Activate.ps1

# Windows (Command Prompt)
venv\Scripts\activate.bat
```

### **Q3: ポート8501が既に使用されている場合**

A: 別のポートで実行してください：
```bash
streamlit run dashboard.py --server.port 8502
```

### **Q4: データを永続的に保存したい場合**

A: 定期的にCSVをダウンロードして保存してください。

### **Q5: ダッシュボードを終了したい場合**

A: ターミナルで `Ctrl+C` を押してください。

## 📞 トラブルシューティング

| 問題 | 解決方法 |
|------|--------|
| テキストが見えない | ブラウザをリロード（F5）してください |
| グラフが表示されない | `pip install --upgrade plotly` を実行 |
| 日本語が文字化けする | ファイルのエンコーディングをUTF-8に設定 |
| データが消える | ブラウザを閉じる前にCSVをダウンロード |

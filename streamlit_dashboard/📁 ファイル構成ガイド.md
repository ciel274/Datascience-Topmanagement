# 📁 ファイル構成ガイド

ダウンロードしたファイルの役割と使い方を説明します。

## 📦 ファイル一覧

```
streamlit_dashboard/
├── dashboard.py              ← メインアプリケーション
├── requirements.txt          ← Python依存パッケージ
├── run.sh                   ← macOS/Linux用実行スクリプト
├── run.bat                  ← Windows用実行スクリプト
├── sample_log.csv           ← サンプル学習ログ
├── README.md                ← 機能説明書
├── SETUP_GUIDE.md           ← 詳細セットアップガイド
├── QUICK_START.md           ← クイックスタートガイド
├── FILE_GUIDE.md            ← このファイル
└── venv/                    ← Python仮想環境（自動生成）
```

## 📄 各ファイルの詳細

### **1. dashboard.py** ⭐ メインアプリケーション

**役割**: Streamlitダッシュボードの本体

**内容**:
- SPI対策ダッシュボードのすべての機能
- 企業・目標設定
- 学習データ入力フォーム
- KPIカード表示
- グラフ・チャート表示
- データ分析ロジック

**使い方**:
```bash
streamlit run dashboard.py
```

**編集が必要な場合**:
- カラーの変更
- 問題マスタの追加
- UI/UXの調整

### **2. requirements.txt** 📦 依存パッケージ

**役割**: Pythonの必要なライブラリを指定

**内容**:
```
streamlit==1.28.1
pandas==2.1.3
numpy==1.26.2
plotly==5.18.0
```

**使い方**:
```bash
pip install -r requirements.txt
```

**編集が必要な場合**:
- 新しいライブラリを追加する場合
- バージョンを更新する場合

### **3. run.sh** 🐧 macOS/Linux用実行スクリプト

**役割**: ワンクリックでダッシュボードを起動

**内容**:
- 仮想環境の作成
- パッケージのインストール
- Streamlitの起動

**使い方**:
```bash
chmod +x run.sh
./run.sh
```

**特徴**:
- 初回実行時に仮想環境を自動作成
- 依存パッケージを自動インストール
- ブラウザを自動で開く

### **4. run.bat** 🪟 Windows用実行スクリプト

**役割**: ワンクリックでダッシュボードを起動（Windows専用）

**内容**:
- 仮想環境の作成
- パッケージのインストール
- Streamlitの起動

**使い方**:
```bash
run.bat
```

または、ファイルをダブルクリック

**特徴**:
- 初回実行時に仮想環境を自動作成
- 依存パッケージを自動インストール
- コマンドプロンプトで実行

### **5. sample_log.csv** 📊 サンプル学習ログ

**役割**: デモ用のサンプルデータ

**内容**:
```csv
日付,問題ID,正誤,解答時間(秒),ミスの原因,学習投入時間(分)
2024-11-01,N-A01,〇,110,-,15
2024-11-01,N-A02,✕,120,時間不足,20
...
```

**使い方**:
1. ダッシュボードを起動
2. サイドバーの「📁 ファイル管理」を展開
3. 「学習ログCSV」に `sample_log.csv` をアップロード
4. ダッシュボードが自動更新

**特徴**:
- 14日間分のサンプルデータ
- グラフやKPIの表示を確認できる
- 実際のデータ形式を参考にできる

## 🎯 シーン別ガイド

### **シーン1: 今すぐ始めたい**

1. **QUICK_START.md** を読む
2. `run.bat` または `run.sh` を実行
3. ダッシュボードが起動

### **シーン2: 詳しく知りたい**

1. **README.md** を読む
2. 全機能を確認
3. 必要に応じて **SETUP_GUIDE.md** を参照

### **シーン3: セットアップでエラーが出た**

1. **SETUP_GUIDE.md** のトラブルシューティングを確認
2. 該当する解決方法を試す
3. 解決しない場合は **README.md** を参照

### **シーン4: ダッシュボードをカスタマイズしたい**

1. **dashboard.py** をテキストエディタで開く
2. 以下の部分を編集:
   - **カラー定義** (17-23行目)
   - **問題マスタ** (25-58行目)
   - **CSS** (77-222行目)

### **シーン5: 新しいライブラリを追加したい**

1. **requirements.txt** を編集
2. 新しいライブラリを追加
3. 以下を実行:
   ```bash
   pip install -r requirements.txt
   ```

## 🔄 ファイルの流れ

```
run.bat/run.sh
    ↓
仮想環境を作成
    ↓
requirements.txt からパッケージをインストール
    ↓
dashboard.py を実行
    ↓
ブラウザでダッシュボード表示
```

## 💾 データの流れ

```
ユーザー入力
    ↓
dashboard.py で処理
    ↓
メモリに保存（ブラウザを閉じるまで）
    ↓
CSV でダウンロード可能
```

## 📊 推奨される使用方法

### **初心者向け**

1. `run.bat` または `run.sh` を実行
2. **QUICK_START.md** に従う
3. サンプルデータで試す
4. 実際のデータを入力

### **中級者向け**

1. **README.md** で全機能を確認
2. 独自のCSVデータを準備
3. カスタマイズが必要な場合は **dashboard.py** を編集
4. 定期的にデータをバックアップ

### **上級者向け**

1. **dashboard.py** のコードを確認
2. 必要に応じてカスタマイズ
3. 新しいライブラリを追加
4. 独自の分析機能を実装
